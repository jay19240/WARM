Class UIMenu

A UI widget displaying a menu with customizable options. It emit 'E_ITEM_FOCUSED' with data { id, index } It emit 'E_ITEM_UNFOCUSED' It emit 'E_ITEM_SELECTED' with data { id, index } It emit 'E_ITEM_UNSELECTED' It emit 'E_UNSELECTED' It emit 'E_CLOSED'
